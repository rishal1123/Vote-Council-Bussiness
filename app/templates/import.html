Expected Excel Format

The Excel file should contain the following columns:

  • # or ID Voter identifier
  • Name Full name (required)
  • G or Gender M/F
  • Age Number
  • Contact Phone number
  • Registered Box Box name (auto-created)
  • Focal Comma-separated names
  • Pledged Y/N
  • Photo Embedded images
Photos can be embedded directly in Excel cells. They will be automatically extracted and linked to voters based on row position.
